Here are short instructions to use the files in the folder.

gui.py
Open the graphical user interface to visualize the simulation of the 4-way intersection, which is configured in the file "complex_variance.py".

complex_variance.py
A python script that includes the configuration of the 4-way intersection.

spawn_complex.npy
Data file that includes the initial positions of the passenger agents. We need this so that the guides are not positioned so that they overlap with the passenger agents.
